Plugin para exibição de mensagens ou imagens em forma de splash.

== Description ==

Este é um simples plugin para exibição de mensagens ou imagens em forma de splash na página inicial.

== Installation ==

1. Envie o arquivo `pr-splash.zip` para o diretório `/wp-content/plugins/`
1. Ative o plugin através do menu Plugins do WordPress
1. Crie sua mensagem através do menu PR Splash do WordPress.
